# 🎯 Number Guessing Game

A beginner-friendly Python game where the computer generates a random number and the player tries to guess it.

## Roadmap

1. Generate a random number.
2. Ask the user to guess the number.
3. Validate the user's input.
4. Tell the user whether the guess is too high or too low.
5. Continue until the correct number is guessed.

## Features

* Random number generation.
* User input validation.
* Error handling using `try-except`.
* Unlimited guessing attempts.
* Feedback after each guess.
* Prevents the program from crashing on invalid input.

## Concepts Used

* Python loops (`while`)
* Conditional statements (`if`, `elif`, `else`)
* Exception handling (`try-except`)
* Random module
* User input
* Comparison operators

## How to Run

```bash
number_guessing_game.py
```

## Example

```text
Enter a number between 1 and 100: 50
Too High

Enter a number between 1 and 100: 25
Too Low

Enter a number between 1 and 100: 37
Congratulations! You guessed the number.
```
